#### CLI (`@vibecoding/cli`)

## [1.0.0] - 2026-07-15

### Added

#### Web Handbook
- Interactive landing page with WebGL Neural Aurora shader effect
- 10-chapter documentation (Basics through Advanced Patterns)
- Prompt Library with 50+ curated prompts and intent detection
- Tool Comparison page (Cursor, Copilot, Claude, v0, Replit, etc.)
- Template Generator with ZIP download
- Project Decision Wizard (9-step interactive framework)
- Prompt Optimizer with side-by-side comparison
- Custom cursor with mix-blend-mode
- Konami code easter egg
- Lazy loading with code splitting (321KB bundle)

#### CLI (`@vibecoding/cli`)
- `vibe init` — Initialize project governance files
- `vibe context --auto` — Smart codebase scanner (15+ frameworks)
- `vibe review` — Code review against project policies
- `vibe optimize` — Prompt optimization with intent detection
- `vibe chat` — Interactive planning session
- `vibe check` — Project validation
- `vibe sync` — Template and prompt library sync

#### VS Code Extension
- Policy sidebar tree view
- Decision (ADR) sidebar tree view
- Stack detection sidebar
- Context preview webview panel
- Initialize, Check, and Optimize commands
