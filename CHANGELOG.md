# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed

#### All packages
- Renamed governance artifact `.cursorrules` → `.iderules` across CLI, web handbook, and VS Code extension. The file is IDE-agnostic; references to Cursor as an IDE remain unchanged.

### Added

#### GitHub Actions
- `vibe-check` composite action in `.github/actions/vibe-check` + dogfood workflow `.github/workflows/vibe-check.yml` — runs `vibe check --strict` and `vibe review --base origin/main` in CI

#### CLI (`@vibecoding/cli`)
- `vibe review --base <ref>` — review files changed since a git ref (e.g. `origin/main`)
- `vibe review --strict` — exit code 1 when errors or warnings are found (CI gating)
- `vibe optimize` AI mode — when `OPENAI_API_KEY` or `ANTHROPIC_API_KEY` is set, the optimized prompt body is rewritten by the LLM (native fetch, no SDK dependencies; `VIBE_PROVIDER` / `VIBE_MODEL` overrides; automatic fallback to local heuristics)
- `vibe init --describe` and `vibe context --describe` — generate `AGENTS.md` from a natural-language project description using OpenAI/Anthropic (native fetch, no SDK dependencies; `VIBE_PROVIDER` / `VIBE_MODEL` overrides; fallback to heuristic generation if the API fails)
- `vibe chat` AI mode — when `OPENAI_API_KEY` or `ANTHROPIC_API_KEY` is set, the plan and optimized prompt are generated by the LLM (native fetch, no SDK dependencies; `VIBE_PROVIDER` / `VIBE_MODEL` overrides; automatic fallback to local heuristics)
- `vibe review --fix` — removes standalone `console.log(...)` statements automatically and prints fix suggestions for the remaining issues

## [1.0.0] - 2026-07-15

### Added

#### Web Handbook
- Interactive landing page with WebGL Neural Aurora shader effect
- 10-chapter documentation (Basics through Advanced Patterns)
- Prompt Library with 50+ curated prompts and intent detection
- Tool Comparison page (Cursor, Copilot, Claude, v0, Replit, etc.)
- Template Generator with ZIP download
- Project Decision Wizard (9-step interactive framework)
- Prompt Optimizer with side-by-side comparison
- Custom cursor with mix-blend-mode
- Konami code easter egg
- Lazy loading with code splitting (321KB bundle)

#### CLI (`@vibecoding/cli`)
- `vibe init` — Initialize project governance files
- `vibe context --auto` — Smart codebase scanner (15+ frameworks)
- `vibe review` — Code review against project policies
- `vibe optimize` — Prompt optimization with intent detection
- `vibe chat` — Interactive planning session
- `vibe check` — Project validation
- `vibe sync` — Template and prompt library sync

#### VS Code Extension
- Policy sidebar tree view
- Decision (ADR) sidebar tree view
- Stack detection sidebar
- Context preview webview panel
- Initialize, Check, and Optimize commands
